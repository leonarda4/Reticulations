# Reticulations

An image processing app that converts images into artistic patterns using various shapes and settings.

## Features

- Upload images or use demo image
- Adjust grid density, contrast, and colors
- Choose from built-in shapes (circle, square, triangle, etc.) or upload custom shapes
- Export processed images

## Getting Started

1. Install Node.js
2. Run `npm install`
3. Run `npm run dev`
4. Open http://localhost:3000

## Technologies

- Next.js
- TypeScript
- Tailwind CSS
- shadcn/ui
- Lucide React
